# lazydotnet

A TUI for managing .NET solutions, inspection, and build management.

## Features
- Tree view of Solution projects.
- Live build output streaming.
- Vim-style navigation.

## Requirements
- .NET 10 (or compatible SDK)

## How to Run
```bash
dotnet run
```
Ensure you are in a directory with a `.sln` file.

## Key Bindings
| Key | Action |
| --- | --- |
| `j` / `Down` | Move Down |
| `k` / `Up` | Move Up |
| `b` | Build Project |
| `q` | Quit |
